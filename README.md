# Сессионный проект по курсу "Операционные системы семейства Unix"

- Host OS: Ubuntu 24.04.3 LTS
- VM OS: Fedora 43
- Виртуализация и вспомогательное ПО:
  - virt-install 4.1.0
  - qemu-img version 8.2.2 (Debian 1:8.2.2+ds-0ubuntu1.16)
  - virsh 10.0.0

## Примечание к настройке сетевого интерфейса
В задании требовалось настроить сетевой интерфейс на использование:
- статического IP-адреса 192.168.1.100/24; 
- шлюза 192.168.1.1;
- DNS-сервера 8.8.8.8. 

В связи действующими настройками сети Host OS, в настройку конфигурации сетевого интерфейса были внесены следующие изменения:
- статический IP-адрес был назначен 192.168.122.100/24
- шлюз был назначен 192.168.122.1.
